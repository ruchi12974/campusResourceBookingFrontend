# Campus Resource Booking System (Frontend)

The frontend interface for the Campus Resource Booking System. Built with **React**, **Vite**, and **Tailwind CSS**. It allows students and faculty to view availability, book resources (labs, halls), and manage their bookings.

## Getting Started Locally

Follow these steps to run the frontend on your local machine.

### 1. Clone the Repository
```bash
git clone [https://github.com/ruchi12974/campusResourceBookingFrontend]
cd frontend 
 ```
###  Install Dependencies 
  **npm install**

### . Environment Setup
Create a .env file in the root directory and add your backend URL. You can use a local backend (localhost:5000) or the live Render backend.
# Example .env file
VITE_API_BASE_URL=http://localhost:5000/api
# OR for live backend:
# VITE_API_BASE_URL=[https://campusresourcebooking.onrender.com](https://your-backend.onrender.com/api)
 
## run the app
npm run dev
